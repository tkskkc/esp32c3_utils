◆ maintainWiFiConnection()

void maintainWiFiConnection ( )

WiFi接続を維持します。

WiFiが未接続の場合、再接続を試行します。接続成功時にはIPアドレス、SSID、MACアドレスをログに出力します。 接続に失敗した場合でも、ループを維持し再起動は行いません。
